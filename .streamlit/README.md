# Streamlit Configuration

Place environment-specific configuration here:

- secrets.toml – production/staging secrets for Streamlit Cloud or local runs.
- credentials.toml – optional Streamlit auth (unused by default).

For quick tunnel demos, log the generated URL each time so the team knows which endpoint to target:

`
## 2025-10-29
- tunnel: https://precisely-prospects-decisions-thinks.trycloudflare.com
- expires: when cloudflared window closes
- notes: used for AI Transactions demo with Streamlit Cloud
`

Use docs/workflow.md for full instructions on updating secrets.
